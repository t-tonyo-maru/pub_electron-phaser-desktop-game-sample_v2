# pri_electron_pub_electron-phaser-desktop-game-sample_v2-sample

Electron + Phaser + TypeScript で構成されたデスクトップゲームのサンプルプロジェクトです。
[node-hid](https://www.npmjs.com/package/node-hid) を利用して、ゲームパッドとの接続を試みるサンプルプロジェクトです。

## クイックスタート

※: Node.js v24.12.0 以上の環境が必要です。

```bash
npm i
npm run start
```
